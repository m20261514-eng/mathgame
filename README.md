# 🌌 수학 차원 대모험 🧭 (Math Dimension Adventure)

> **"새벽의 빛을 따라 차원의 문을 열고, 수학 유물을 수집하는 전설의 탐사 대원이 되어보세요!"** >
> 초등학생 및 학습자들이 곱셈, 역곱셈, 나눗셈을 게임처럼 즐겁게 학습할 수 있도록 동기를 부여하는 보상형 인터랙티브 학습 플랫폼입니다.

---

## 🚀 프로젝트 개요
* **개발 목적:** 딱딱하고 지루한 수학 연산 학습을 '차원 탐험'과 '유물 도감 수집'이라는 게임 요소를 결합하여 학습 몰입도를 극대화합니다.
* **핵심 가치:** 핀번호 기반의 간편한 데이터 보존, 화려하고 몰입감 넘기는 다크 테마 UI/UX, 성취감을 자극하는 등급별 도감 시스템.

---

## ✨ 주요 기능

### 1. 기본 기능 (Core Features)
* **🔐 핀번호 기반 간편 로그인:** 복잡한 가입 절차 없이 학년-반-번호 등의 고유 핀번호 입력만으로 개인 학습 데이터와 즉시 연동됩니다.
* **🎮 3대 수학 차원 게이트:** 사용자의 선택에 따라 곱셈, 역곱셈, 나눗셈 학습 페이지로 안전하게 라우팅합니다.
* **📊 3대 테마별 도감(아카이브):** 초원, 바다, 우주 테마의 도감을 제공하며, 수집 여부에 따라 카드가 컬러(`획득`) 또는 회색 자물쇠(`미확인`)로 실시간 시각화됩니다.

### 2. 심화 및 기술적 특징 (Advanced Features)
* **💾 Serverless 데이터 영속성:** 별도의 무거운 DB 없이 `student_data/[핀번호].json` 구조의 파일 시스템과 `st.session_state`를 연동하여 유저의 자산(Gold) 및 수집 데이터를 안전하게 관리합니다.
* **🎨 CSS 하이재킹 테마:** Streamlit의 기본 UI 제약을 넘어 대형 네온 버튼, 크로매틱 텍스트 애니메이션(`dimension-shift`), 다크모드 입력창 고정 패치 등 하이엔드 게임 커스텀 스타일을 구현했습니다.
* **🖼️ Base64 이미지 임베딩:** 배포 환경에 따른 경로 깨짐을 원천 차단하기 위해, 로컬 배경 이미지를 Base64 바이너리로 변환하여 CSS 레이어에 직접 주입했습니다.

---

## 🛠️ 기술 스택 (Tech Stack)
* **Language:** Python 3.x
* **Framework:** Streamlit
* **Data Management:** JSON (File-based DB)
* **Frontend Customization:** HTML5, CSS3 (Keyframe Animations)
