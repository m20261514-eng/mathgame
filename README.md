# mathgame
# 🥚 수학 게임 대모험 (Math Game Adventure) 🎲

스트림릿(Streamlit)을 기반으로 제작된 아기아기하고 재치 넘치는 초등학생 맞춤형 수학 학습 웹 어플리케이션입니다. 학생들이 지루해하는 역곱셈과 나눗셈 연산을 게임 형식을 통해 재미있게 훈련할 수 있도록 설계되었습니다. 

문제를 풀어 💰골드를 모으고, 모은 골드로 🥚신비의 알을 부화시켜 나만의 동물 도감을 완성해 보세요!

---

## ✨ 주요 특징 (Features)

* **통합 세션 시스템:** 메인 로비, 역곱셈 훈련장, 나눗셈 훈련장이 실시간으로 `골드(Gold)`와 `동물 도감(Collection)` 데이터를 안전하게 공유합니다.
* **비밀 확률형 뽑기 (Gacha):** 가챠 등급의 확률(일반 70%, 희귀 25%, 전설 5%)은 백엔드에서 정밀하게 작동하지만, 학생들의 사행성 조장을 방지하기 위해 겉으로는 확률 수치가 절대 드러나지 않습니다.
* **완벽한 모바일 반응형:** 교실에서 태블릿이나 스마트폰으로 접속해도 3x3 입력 키패드와 바둑판식 도감이 깨지지 않고 쾌적하게 유지됩니다.
* **친절한 힌트 시스템:** 오답 발생 시 곱셈과 나눗셈의 역산 관계를 직관적으로 이해할 수 있는 시각적 힌트를 제공합니다.
